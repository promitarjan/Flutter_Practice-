# Digital Business Card

A beginner Flutter UI practice project that demonstrates how to build a clean and professional digital visiting card screen.

This app displays a profile image, name, designation, contact details, social icons, and a card-based layout with a light purple professional theme. It was created as part of my Flutter practice journey to learn widget tree structure, layout basics, text styling, reusable widgets, and simple composition.

## Features

- Profile image using `CircleAvatar`
- Name and designation section
- Contact details card
- Social icons section
- Custom dot divider
- Scrollable layout
- Reusable widgets
- Clean card-based UI
- Light purple professional theme

## Learning Goals

- Understand the Flutter widget tree
- Practice layout building with `Column`, `Row`, and `Padding`
- Learn how to create reusable widgets
- Use `Card` to group related content
- Improve text styling and spacing
- Handle overflow with `SingleChildScrollView` and `Expanded`
- Build a simple and readable UI for a beginner project

## How to Run the Project

1. Clone or open the project in VS Code or Android Studio.
2. Get dependencies:

	```bash
	flutter pub get
	```

3. Run the app on an emulator, simulator, or connected device:

	```bash
	flutter run
	```
